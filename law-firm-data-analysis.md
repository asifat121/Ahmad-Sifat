# Law Firm Data Analysis (ETL & Reporting)

## Overview
This project demonstrates an end-to-end **data transformation and analysis workflow** using real-world, messy datasets from **1,100+ law firms**. The focus is on converting unstructured and semi-structured data into clean, analysis-ready tables that support business insights and reporting.

## Business Value
- Improved usability of raw legal datasets for downstream analysis
- Reduced manual data cleaning through automated transformations
- Enabled faster insight generation for reporting and decision-making

## Responsibilities
- Designed and executed ETL workflows using Excel Power Query
- Cleaned inconsistent and malformed text data at scale
- Transformed nested, pipe-delimited fields into flat analytical tables
- Prepared final datasets for reporting and visualization

## Tools & Technologies
- Excel
- Power Query
- ETL & Data Wrangling Techniques

## Workflow
1. Ingest raw CSV datasets
2. Clean and standardize text fields
3. Normalize nested personnel data
4. Output structured tables for analysis

## Key Takeaway
Demonstrates strong **data cleaning, ETL, and analytical thinking**, essential for remote data analyst roles.
